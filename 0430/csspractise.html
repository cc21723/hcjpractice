優勢

Strengths

  • 熟悉 HTML/CSS/JavaScript 與現代框架（如 React、Vue）
  • 具備良好 UI/UX 敏感度與前端設計思維
  • 與設計師、後端工程師協作能力強
  • 技術變動快，學習曲線陡但能快速成長

劣勢

Weaknesses

  • 技術更新頻繁，需不斷學習以保持競爭力
  • 專案中容易被視為「視覺實作」角色而非主導開發
  • 不同瀏覽器、裝置兼容性問題造成額外工作量
  • 與後端或資安等系統層理解可能較弱

機會

Opportunities

  • Web 技術進化（PWA、WebAssembly、AI 前端工具）帶來新可能
  • 遠端工作、自由接案的彈性越來越高
  • 移動裝置與 WebApp 趨勢穩定成長
  • 有機會跨足產品設計、互動動畫、UX 領域

威脅

Threats

  • 前端框架推陳出新，造成技術棄用壓力
  • AI 工具（如 GitHub Copilot）正在自動化部分程式撰寫
  • 職場競爭激烈，作品與面試表現變得更關鍵
  • 工程師市場供需快速波動，有外包取代風險
